현재 Vision Auto v2 매크로 시스템은 기능적 구조가 이미 명확히 잡혀 있으므로,
이제 **“매크로 에디터 UI/UX 설계 방안”**을 중심으로,
프로토타입 → 정식 설계 단계로 전환할 수 있는 방향을 제안드릴게요.

⸻

🎯 목표

“매크로를 직관적으로 추가, 편집, 관리할 수 있는 통합형 에디터”
사용자가 ‘한눈에 구조를 이해하고, 한 번 클릭으로 제어할 수 있는’ 환경 구현.

⸻

🧱 전체 레이아웃 구조 제안

────────────────────────────────────────────
| [좌측] 매크로 목록 패널                   |
|--------------------------------------------|
| [중앙] 액션 에디터 & 타임라인              |
|--------------------------------------------|
| [우측] 속성(파라미터) 설정 / 프리뷰        |
────────────────────────────────────────────

1️⃣ 좌측 패널 — 매크로 탐색 / 관리

위치: 화면 좌측 20~25%
역할: 매크로들의 생성, 탐색, 선택, 복제, 삭제

구성 요소

요소	설명
🔍 검색창	매크로 이름 검색 / 필터
📋 매크로 리스트	각 항목: 이름, 액션 수, 생성일
➕ 새 매크로 버튼	새 매크로 생성 (빈 시퀀스 생성)
🗑️ 삭제 / 복제 버튼	다중 선택 후 일괄 작업
▶ 실행 버튼	선택된 매크로 바로 실행

UX 포인트
	•	드래그 앤 드롭 정렬 지원
	•	“최근 편집한 매크로” 자동 상단 고정
	•	더블클릭 시 에디터 중앙으로 로드

⸻

2️⃣ 중앙 패널 — 액션 에디터 & 타임라인 뷰

위치: 중앙 50~60%
역할: 액션을 시각적으로 편집 / 순서 제어 / 그룹화

기본 구성

UI 영역	설명
🎞️ 타임라인형 리스트	액션 순서가 시각적으로 드러나는 구조
⬇ 드래그 핸들	순서 변경 가능
🧩 액션 카드	각 액션의 요약 (아이콘 + 주요 파라미터)
✏️ 편집 버튼	클릭 시 우측 속성 패널 활성화
❌ 삭제 버튼	즉시 제거
⏱️ 실행 포인트 마커	실행 중일 때 하이라이트 애니메이션 표시

UX 포인트
	•	액션들을 드래그로 순서 변경
	•	Ctrl/Cmd+클릭 → 다중 선택 후 일괄 편집
	•	실시간 미리보기: 현재 선택된 액션이 디바이스 화면 미니뷰에서 하이라이트됨

⸻

3️⃣ 우측 패널 — 액션 속성 / 미리보기

위치: 화면 우측 20~25%
역할: 선택된 액션의 세부 속성 편집 및 프리뷰 제공

구성 예시

항목	설명
액션 타입	tap, swipe, scroll, input, …
주요 파라미터	예: x, y, duration, text
테스트 실행 버튼	해당 액션만 개별 실행
🔄 미리보기	미니 디바이스 뷰에서 시각화 (예: tap 위치 표시)
💾 변경 적용 버튼	즉시 반영 후 ActionStore 업데이트

UX 포인트
	•	즉시 반응형 미리보기
	•	입력 검증(Validation): 좌표, 시간, 텍스트 유효성 자동 체크
	•	기본값 자동 완성: 마지막 사용값 기억

⸻

🧭 상단 툴바 (전역 제어)

기능	설명
▶ 실행	현재 매크로 전체 실행
⏸️ 중지	실행 중단
💾 저장	매크로 저장 / 이름 변경
📂 불러오기	JSON 파일 가져오기
📤 내보내기	매크로 JSON 파일로 저장
⚙️ 설정	ADB 경로, 디바이스 연결 관리
🧪 테스트 모드	단계별 실행 디버그 모드 진입


⸻

💡 상호작용 UX 플로우

시나리오	UI 동작 흐름
1. 새 매크로 생성	“+ 새 매크로” → 이름 입력 → 빈 타임라인 생성
2. 액션 추가	“+ 액션” 버튼 → 타입 선택 → 중앙에 새 카드 삽입
3. 액션 편집	카드 클릭 → 우측 패널에서 속성 수정 → 실시간 반영
4. 실행 중 표시	현재 실행 중 액션 카드 하이라이트 + 스크롤 자동 이동
5. 녹화 모드	“녹화 시작” → 터치/입력 자동 기록 → 타임라인에 즉시 반영
6. 매크로 저장	자동으로 localStorage 및 JSON 파일에 저장

🧩 시각 디자인 방향

요소	제안 스타일
색상	다크 모드 기반 (블루/그레이 톤)
