# Vision Auto v2 - 테스트 체크리스트

## 테스트 실행 방법

### 자동화 테스트
```bash
# 단위 테스트
npm test

# E2E 테스트
npx playwright test

# 커버리지 리포트
npm run test:coverage
```

### 수동 테스트
아래 체크리스트를 따라 테스트합니다.

---

## 1. 앱 시작 및 기본 UI

### 1.1 앱 시작
- [ ] `npm start`로 앱이 정상 실행됨
- [ ] 콘솔에 에러 없음
- [ ] 메인 윈도우가 표시됨

### 1.2 UI 레이아웃
- [ ] 3개 패널 구조가 보임 (시나리오 | 화면 | 액션)
- [ ] 헤더/툴바가 정상 표시됨
- [ ] 로그 패널이 하단에 표시됨

### 1.3 초기 상태
- [ ] "저장된 시나리오가 없습니다" 메시지 (첫 실행 시)
- [ ] 액션 목록이 비활성화 상태
- [ ] "새 시나리오" 버튼이 보임

## 2. 시나리오 관리

### 2.1 새 시나리오 생성
- [ ] "새 시나리오" 버튼 클릭
- [ ] 시나리오 이름 입력 가능
- [ ] 파일명 프리뷰가 표시됨
- [ ] 생성 후 편집 모드로 전환됨
- [ ] 액션 목록이 활성화됨

### 2.2 시나리오 저장
- [ ] 저장 버튼 클릭 시 저장됨
- [ ] "저장됨" 피드백 표시
- [ ] 새로고침 후에도 데이터 유지됨

### 2.3 시나리오 로드
- [ ] 시나리오 목록에서 클릭하여 로드
- [ ] 이름, 설명, 액션 모두 복원됨
- [ ] 저장 안된 변경사항 경고 표시 (있을 경우)

### 2.4 시나리오 삭제
- [ ] 삭제 버튼 클릭
- [ ] 확인 다이얼로그 표시
- [ ] 삭제 후 목록에서 제거됨
