4️⃣ 연결 상태 관리 로직 (iSAP 예시)

주기적으로 TCP 연결 여부를 확인해 UI 갱신

loop every 3s:
    try:
        socket.connect(target_ip, 20000, timeout=1s)
        state = CONNECTED
    catch:
        state = DISCONNECTED

	•	Interval: 2~5초
	•	Timeout: 1~2초
	•	연속 3회 실패 시: “끊김” 전환
	•	마지막 성공 시각 표시: “마지막 응답: 2초 전”
